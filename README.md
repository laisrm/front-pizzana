## Descrição 

Projeto completo Fullstack para página de pizzaria.  <br />
A API consumida neste projeto é própria e foi criada usando MySQL Workbench, Postman e TypeScript.

 <br />
 
⚙️
## Bibliotecas e Frameworks

▪ React JS
▪ React Hooks
▪ React Router
▪ Axios
▪ API própria
▪ Styled-components
▪ Material UI
